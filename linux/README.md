# Linux useful commands

## Show file content from terminal

- Show all file content:

    ```bash
    cat [file-path]
    ```

- Show the first `n` lines:

    ```bash
    head -[n] [file-path]
    ```

- Show file content with line number:

    ```bash
    cat -n [file-path]
    ```

- Count number of lines in text file:

    ```bash
    wc -l [file-path]
    ```
